Summary

  • To make it look more like a map, used the background of image captured the map in the pdf file.
    On top of that, created rectangles that were sized and positioned, then grouped into 'map-groups'.

  • Using 'function()'' method, which was used in Lab1, let the color was changed when the mouse was placed on each square.
    After defining element as target, set the value of 'unselected_colour'by adding a new property using the given namespace or name.
    'selected_color' was changed the value of colour property .

  • 'document.getElementById()' method finds an element with an 'id' property that matches a given string and returns an 'element' object that represents it.
    Using this method, defined 'map-group' as 'map_obj' and added 'addEventListener()' to define the function to be called to the event.
    Therefore, when mouseover the square, the color changes from gray to blue.

  • Using 'tooltip' function, I can popup the info about building.
